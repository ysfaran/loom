# Packages Workspace

This folder contains Loom workspace packages.

Current package:
- `@loom/core` for shared scanning, validation, and rendering primitives.
- `@loom/renderer-vite-react` for React-based build and dev rendering.
- `@loom/renderer-astro` for Astro-based build and dev rendering.

Planned packages:
- `@loom/*` plugin packages (search, ADR support, and more).
